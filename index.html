10

AWS Terraform High Availability

Arquitectura modular de alta disponibilidad en AWS implementada con Terraform. Diseño escalable con recursos optimizados: múltiples VPCs con subnets en diferentes Availability Zones, Application Load Balancer con Auto Scaling Groups para EC2, NAT Gateway para acceso seguro, EFS para almacenamiento compartido y SES para notificaciones de email. Implementación 100% modularizada para evitar acoplamiento y facilitar reutilización y mantenimiento. (Plus principal, no comprar una base de datos adminsitrada, si no integrar la base de datos como imagen en una EC2 aparte sin ASG y conectada a el EFS para alojar sus volumenes por si ocurre una perdida de la instancia.)

Terraform AWS EC2 ALB Auto Scaling VPC NAT Gateway EFS SES Modules
